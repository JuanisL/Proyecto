Imagenes

La etiqueta <img> es usada para insertar una imagen.Contiene sólo atributos y no tiene una etiqueta de cierre.
La URL de la imagen (dirección) puedes ser definida usando el atributo src.

La sitaxis HTML de una imagen es:

<img src="imagen1.jpg"/>

Para que el explorador web pueda desplegar una imagen, se necesita colocar dentro de comillas la ubicación de la imagen como valor del atributo src.
Cuando la imagen no se pueda mostrar, el atributo alt especifica un texto alternativo que describe el contenido de la imagen.

Ejemplo:

<img src="imagen1.jpg" alt="Pintura geometrica"/>

Hipervinculos

La etiqueta <a>

Los hipervinculos son una parte fundamental de cualquier pagina web.

Se puede agregar hipervinculos a textos o imagenes que permitan al usuario hacer clic en ellos para luego ser redirigidos a otro archivo o pagina web.En HTML, los hipervinculos son definidos usando la etiqueta <a>

El atributo hrefse usa para definir la dirección de destino del hipervinculo:

El atributo target especifica donde abrir el documento vinculado.

Dandole un valor _blank al atributo abrirá el hipervinculo en una nueva ventana o pestaña.

Ejemplo:

<a href="http://www.google.com.mx"target="_blank">Google</a>
